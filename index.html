1 O que é ECMA?

  • European Computer Manufacturers Association
  • Associação dedicada à padronização de sistemas de informação
  • O JavaScript foi criado para o navegador Netscape (Brendan Eich - 1995)

2 Criação do ECMAScript

  • Necessidade de uma linguagem padrão para navegadores
  • “General-purpose programming language”
  • ECMAScript é uma padronização do JavaScript
  • ECMA + JavaScript = ECMAScript

3 Novidades da ES6

3.1 Default parameter

  • Parâmetros de funções têm undefined como valor default

  • Antes da ES6:

    
                                    function exibeValor(x, y){
                                        x = x | 0;
                                        y = y | 0;
                                        console.log("1º valor: " + x);
                                        console.log("2º valor: " + y);
                                    }
                                    exibeValor();

Novidades da ES6

3.1 Default parameter

  • Depois da ES6:

    
                                    function exibeValor(x = 0, y = 0){
                                        console.log("1º valor: " + x);
                                        console.log("2º valor: " + y);
                                    }
                                    exibeValor(1,2);

Novidades da ES6

3.2 Template strings

  • Antes da ES6:

    
                                    function soma2(x = 0, y = 0){
                                        console.log("A soma de " + x + " e " + y + " é igual a " + (x+y));
                                    }
                                    soma2(1,2);

  • Depois da ES6:

    
                                    function soma2(x = 0, y = 0){
                                        console.log(`A soma de ${x} e ${y} é igual a ${x+y}`);
                                    }
                                    soma2(1,2);

Novidades da ES6

3.3 Destructuring

  • Permite extrair dados de arrays ou objetos em variáveis distintas

  • Exemplos (Arrays):

    
                                    const lista = [1,2,3,4,5];
                                    const [a,b] = lista;
                                    console.log(a);   // 1
                                    console.log(b);   // 2

Novidades da ES6

3.3 Destructuring

  • Exemplos (Objetos):

                  
                                    const carro = {
                                        cor: 'black',

5 Hoisting

  • No JavaScript as declarações são movidas para o topo
  • É uma boa prática declarar variáveis no topo
  • Só funciona com declarações var

5 Hoisting

  • Exemplos:

    
                                    vDefault = 'nada';
                                    function exibeValor(x = vDefault, y = vDefault){
                                        console.log("1º valor: " + x);
                                        console.log("2º valor: " + y);
                                    }
                                    exibeValor();
                                    var vDefault;

6 var, let e const

  • var tem escopo de função e let/const tem escopo de bloco:

    
                                function somaX(a, condicao = true) {
                                    if (condicao) {
                                        var x = 2;
                                    }
                                    return a + x;
                                }
                                console.log(somaX(2));

7 for, forEach e for... of

  • O comando for fornece acesso ao índice do array, diferente dos outros:

    Exemplo com for:

    
                                    function maior(lista) {
                                        let maiorValor;
                                        if(lista === undefined || lista.length == 0)
                                            return null;
                                        maiorValor = lista[0];
                                        for(let i = 1; i < lista.length; i++){
                                            if(maiorValor < lista[i])
                                                maiorValor = lista[i];
                                        }
                                        return maiorValor;
                                    }
                                    console.log(maior([1,20,35,-100,47,32]));

7 for, forEach e for... of

  • Exemplo com forEach:

    
                                    function maior(lista) {
                                        let maiorValor;
                                        if(lista === undefined || lista.length == 0)
                                            return null;
                                        maiorValor = lista[0];
                                        lista.forEach(elemento => {
                                            if(maiorValor < elemento)
                                                maiorValor = elemento;
                                        });
                                        return maiorValor;
                                    }
                                    console.log(maior([1,20,35,-100,47,32]));

7 for, forEach e for... of
